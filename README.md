# Pet_Adoption_System-using-Hibernate
Pet Adoption Management System is a console-based Java application using Hibernate ORM and Oracle Database. It supports CRUD operations for pets and adoptions, automatic ID generation, and validation for pet availability and medical status. The project demonstrates ORM, Hibernate session management, and transaction handling.
